Ch2: Programming Basics

  • You can make single line comments with "//", or you can create paragraph comments with "/* */".
  • Each statment is compleated either by a new line or a ";" at the end. You can even have two statments on the same line when the ";" is used.
  • Variables are deturmend by three key words:
    1. var makes a global variable.
    2. const prevents the variable from ever being changed.
    3. let creates a variable that is only accessable withen its block.
  • In strings, use "\" to include symbols are create commands.

    • Ch3: Arrays, Logic, and Loops

  • Arrays names are created the same way as variables but are set equal to "[]".
    1. To add to a list, you just place the number position in the list brackets, "[]", and set it equal to the item. Keep in mind you can overwrite already existing items this way.
    2. Removing items from the list uses the keyword "delete" infront of the list, then stating which item you want removed: delete list[3];
    3. Destructuring an aray alows you to create multiple variables using an aray: let [x,y] = [1,2];
  • If statments are created using the words if, else if, and else, ending with "{}" with the code inside.
  • Switch operators are used to test conditions. There are 4 key words:
    1. switch - starts the statement and accepts what is being checked: switch (variable){}
    2. case - located inside the switch brackets and checks if the switch parameter equals something: case "yes":
    3. break - ends the statement
    4. default - if no criteria is met. This is usually found at the ending of the statement.
  • While loops will repete code until while a condition is met: while (condition){}
  • do...while loops will start the code before checking for the condition: do {} while (condition)
  • For loops will repet code untill a condition is satisfied: for (initialization; condition; after) {}
