Ch 8: Forms

Form controls

forms are made up of 'form' elements. Forms contain controls such as input fields, select menus and buttons. Of these we can submit it serverside where the information would be processed.

Accessing Form Elements

Forms can be accessed in javascript using document.forms[#]. Every form gets colected into this list in the order they appear in the markup. However, it may be eaiser to use the forms name rather than to know their order number. document.forms['name']. Keep in mind that these forms are set into variables.

Form Properties and Methods

  • form.submit() method submits the form automatically.
  • type='submit' attribute on a button or input requires manual submission.
  • form.reset() method resets all from controls back to their initial values. This can also be done with a button that has a type='reset' attribute. However, reset buttons are fround uppon
  • form.action = '/an/other.url' sends the information to a different URL to be processed.

    OOP is made up of three concepts:

  • Encapsulation-The user can eaisly run the program without needing to know the "inner workings of the code."
  • Polymorphism-Each object function in the same way and can be compatable with other code.
  • Inheritance-Instead of creating something similar to an already existing object, implemet it into you new code.

    • Constructor functions

    A function that defines the properties and methods of an object. It implements the keyword 'this' to signify that the variable will be returned by the constructor function. This function will be used for the constructor property, which all objects have.

    Static Methods

    If you put the word 'static' infront of the function, it allows you to call it from the class directly rather than by instances of the class. However, these functions are not available to instances of the class.

    Prototype

    This is a handy way for accessing and/or changing properties and functions in a class.

  • class.prototype.variable = 'value' - Sets its variable value.
  • class.prototype.function = function(){code} - Sets its function.
  • Object.getPrototypeOf(object_name) - Finds the prototype of an object.
  • class.prototype.isPrototypeOf(object_name) - returns a boolean to check if its the prototype of an instance
  • object_name.hasOwnProperty('') - checks to see if an attribute is unique or inherited.
