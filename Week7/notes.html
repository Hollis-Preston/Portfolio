Function Properties and Methods

  • function.length - returns the number pr parameters the function has.
  • function.call(object) - incokes the function, providing each object as an argument
  • function.apply(null, [#]) - the arguments of the function are provided as an array

    • You can create custom properties:

  • function.description = 'This is the description of the function'

    • Invoked function expressions

    An anonymous function that is invoked as soon as it's defined. This is achieved by placing parentheses at the end of the function definition.

  • (function(){code})();

    • Self-defining functions

    The ability for the function to not anly call itself, but define itself, and even redefineitself. This is done by assigning an anonymous function to a variable that has the same name as the function.

  • function name(){code; name = function(){code}}

    • This will run whatever code it contains the run the code of the function withen the next few times the function is called.

    Recursive functions

    A function that invokes itself until a certain condition is met.

  • function name(x){if(x===0){return 1} else{return name(x-1)} }

    • Callbacks

    Functions passed to other functions as arguments and then invoked inside the function they are passed to.

    Promises

    Represents the future result of an asynchronous operation.
