Chapter 8: Transforms and Transitions

Transforms

the transform property lets you translate, rotate, scale, and/or skew any element on the page. This gives you uprecedented controll over many more aspects of the elements appearance then possible in CSS.

  • transform: translate(x, y)-This function allows you to shift elements on the x or y axis.
  • transform: scale(x, y)-Scales an element by the defined factors horizontally the vertically.
  • transform: rotate(deg)-rotates an element around the point of orgin by a specified angle.
  • transform: skew(x_deg, y_deg)- apecifies a skew along the x and y axes.

  • Transitions

    Animation. While it can be done trhough javaScript, native CSS transitions generally require less client-side processing and appear smoother.

  • transition-property: transform
  • transition-duration: 0.2s-Sets how long the transition will take.
  • transition-timing-function: command-Lets you controll the pace of the transition in even more granular detail.

    • commandes include: ease, linear, ease-in, ease-out, ease-in-out, steps(#, start/end)

  • transition-delay: time-Introduces a delay before the transition begins.
