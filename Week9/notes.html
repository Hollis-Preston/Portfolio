Chapter 09: The Window Object

The Browser Object Model

BOM is a collection of properties and methods that contain information abou the browser and computer screen.

BOM only makes sence to run in a browser environment

  • window.variable - In a browser encironment, the global object is the window object.
  • Number.parseInt(#) - makes a decimal into an int
  • Number.isNaN(#) - checks to see if number is int or not.
  • alert(), confirm(), and prompt() are not part of the ECMAScript standard, although all major browers support them as methods of the window. object: window.alert().

    • Finding out browser information

    The window object has a number of properties and methods that provide information about the user's browser.

    The window object has a navigator property that returns a reference to the Navigator object. The Navigator object contains information about the browser being used. Its userAgent property will return information about the browser and operating system being used.

  • window.location.hrf - returns the full URL as a string.
  • window.location.protocol - returns a string describing the protocol used (http, https, etc.)
  • window.location.host - returns a string describing the domain of the current URL and the port number.
  • window.location.hostname - returns a string describing the domain of the current URL.
  • window.location.port - returns a string describing the port number.
  • window.location.pathname - returns a string of the path that follows the domain.
  • window.location.search - returns a string that starts with a "?" followed by the query string parameters.
  • window.location.hash - returns a string that starts with a "#" followed by the fragment identifier.
  • window.location.origin - returns a string that shows the protocol and domain where the current page originated from

    • Browser history

  • window.history.lenght - shows how many pages have been visited before arriving at the current page.
  • window.history.go(#) - used to go to a specific page, where 0 is the current page.
  • window.history.forward()/back() - navigates forwards and backwards by one page respectively.

    • Controlling windows

  • window.open('URL', 'SitePoint', 'width=#, height=#, resizable=yes') - used to open a new window.
  • window.close() - closes a window
  • window.moveTo(#,#) - moves a window to an x,y location.
  • window.resizeTo(#,#) - resizes a window
  • window.screen.height/width - gives the height or width of the screen in pixels.

    • Cookies

    Small filse that are saved locally on a user's computer. Alowes a webpage to remember things about previous visits.

  • document.cookie = 'key=value' - assigning a cookin in javaScript.
  • document.cookie: - reading cookies.
  • document.cookie = 'key=value; path=/' - any page in the root directory can read the cookie
  • document.cookie = 'key=value; domain=website.com' - same as above but with a domain name.
  • document.cookie = 'key=value; secure' - ensures its only transmitted over a secure HTTPS network:
  • document.cookie = 'key=list; expires=Thu, 01 Jan, 2021 00:00:01 MST' - deletes a cookie at a given time. Can delete instently if the date is already past.

    • Timing functions

  • window.setTimeout( () => alert("Times Up!"), 3000) - alerts after a certain amount of milliseconds
  • window.clearTimeout(#) - if the number given from the previous code is imputed, then the alert is prevented.
